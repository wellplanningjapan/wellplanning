# Well Planning - ホテルライクな家具・家電ECサイト

## プロジェクト概要

Well Planningは、ホテルライクな暮らしをコンセプトにした家具・家電のオンラインショップです。新生活を始める方や引越しを考えている方に、厳選された商品をパッケージで提供し、初日から快適で美しい暮らしを実現します。

## 主な機能

### ✅ 実装済み機能

1. **トップページ**
   - ヒーローセクション（メインビジュアル）
   - おすすめパッケージ一覧（3種類）
   - ブランド紹介セクション
   - レスポンシブデザイン対応

2. **パッケージ管理**
   - パッケージデータベース（3パッケージ登録済み）
   - パッケージ詳細表示
   - 価格表示
   - 含まれる商品数の表示

3. **ショッピングカート機能**
   - カートへの商品追加
   - 数量変更
   - 商品削除
   - 合計金額の自動計算
   - 配送料計算（10万円以上で送料無料）
   - 消費税計算（10%）
   - ローカルストレージでカート保持

4. **決済・チェックアウト**
   - 4ステップのチェックアウトフロー
     - ステップ1: お客様情報入力
     - ステップ2: 配送情報入力
     - ステップ3: 支払い方法選択
     - ステップ4: 注文確認
   - フォームバリデーション
   - 注文完了モーダル
   - 注文番号の自動生成

5. **UIデザイン**
   - 洗練されたベージュ・ブラウン系カラーパレット
   - Font Awesome アイコン統合
   - Google Fonts（Noto Sans JP、Playfair Display）
   - スムーズなアニメーション
   - モバイルファーストレスポンシブデザイン

6. **多言語対応（日本語/英語）**
   - ヘッダーに言語切り替えボタン
   - ワンクリックで日本語⇔英語を切り替え
   - 選択した言語をローカルストレージに保存
   - 全ページ・全コンテンツが対応
   - 商品名・説明も自動翻訳

7. **SNS連携**
   - Instagram QRコード
   - YouTube QRコード
   - Spotify QRコード
   - フッターに3つのQRコードを表示
   - スマホでスキャンして簡単フォロー

## 機能エントリーポイント（URI）

### メインページ
- **トップページ**: `index.html`
  - ヒーローセクション
  - パッケージ一覧: `index.html#packages`
  - ブランド紹介: `index.html#about`

### 商品関連
- **商品詳細**: `product.html?id={商品ID}`
  - 例: `product.html?id=product_1`

### カート・決済
- **ショッピングカート**: `cart.html`
- **チェックアウト**: `checkout.html`

### API エンドポイント
- **商品一覧取得**: `GET tables/products?limit=100`
- **商品詳細取得**: `GET tables/products/{商品ID}`
- **パッケージ一覧取得**: `GET tables/packages?limit=100`
- **パッケージ詳細取得**: `GET tables/packages/{パッケージID}`

## データモデル

### 商品テーブル（products）
```
- id: 商品ID（必須）
- name: 商品名
- category: カテゴリー（furniture/appliance）
- subcategory: サブカテゴリー
- price: 価格（税込）
- description: 商品説明
- image_url: 商品画像URL
- stock: 在庫数
- featured: おすすめ商品フラグ
```

### パッケージテーブル（packages）
```
- id: パッケージID（必須）
- name: パッケージ名
- description: パッケージ説明
- price: 価格（税込）
- items: 含まれる商品IDの配列
- image_url: パッケージ画像URL
- featured: おすすめパッケージフラグ
```

## セットアップ方法

1. **Publishタブ**から「Publish」ボタンをクリックしてサイトを公開
2. 公開されたURLにアクセス
3. 商品データは既に登録済みのため、すぐに利用可能

**重要**: 開発環境（プレビュー）では、APIへのアクセスがブロックされる場合があります。**必ずPublishタブから公開**してください。公開後はすべての機能が正常に動作します。
